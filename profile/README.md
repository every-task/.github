## 3. 사용 기술
- Back End : Java 17, Spring Boot 3.x.x, Spring Security, Spring Data JPA, Query DSL
- Front End : React, MUI
- DB : MySQL, Mongo DB
- Messaging : Kafka, Kafka Connect
- Infra : Docker, Kubernetes Nginx, GCP
- Storage : Firebase
- CI/CD : Github Action
- 3rd Party : Open AI API
